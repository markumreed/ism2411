Weekly Quiz ★ Midterm-Eligible

Quiz questions repeat on the exam verbatim or with minor variation. Click each answer to reveal it.

Q1 · Multiple Choice
What keyword defines a function in Python?
  • a) function
  • b) define
  • c) def
  • d) fun
Answer
def
Q2 · Code Trace
What is the output?
def greet(name):
    return 'Hello, ' + name
print(greet('Maria'))
Answer
Hello, Maria
Q3 · Code Trace
What does tax(200, 0.1) return?
def tax(price, rate):
    return price * rate
print(tax(200, 0.1))
Answer
20.0
Q4 · Multiple Choice
What is a function parameter?
  • a) The return value
  • b) A variable that receives a value when the function is called
  • c) The function name
  • d) A type of loop
Answer
A variable that receives a value when the function is called
Q5 · Code Trace
What error does this produce and why?
def add(a, b):
    return a + b
print(add(5))
Answer
TypeError — add() requires 2 arguments but only 1 was given
Q6 · Multiple Choice
Which best describes a traceback?
  • a) A list of all variables
  • b) An error message showing where and why a program failed
  • c) A way to undo code changes
  • d) A backwards loop
Answer
An error message showing where and why a program failed
Q7 · Code Trace
What is the output?
def double(x):
    return x * 2
result = double(double(3))
print(result)
Answer
12 — double(3)=6, double(6)=12
