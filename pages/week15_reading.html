Q1 · Multiple Choice
Which matplotlib function creates a bar chart?
  • a) plt.line()
  • b) plt.bar()
  • c) plt.scatter()
  • d) plt.hist()
Answer
plt.bar()
Q2 · Code Trace
What does this code produce?
import matplotlib.pyplot as plt
plt.hist(df['sales'], bins=10)
plt.title('Sales Distribution')
plt.show()
Answer
A histogram of sales with 10 bins and the title 'Sales Distribution'
Q3 · Multiple Choice
When is a line chart the best choice?
  • a) Comparing categories
  • b) Showing a distribution
  • c) Showing a trend over time
  • d) Showing two unrelated variables
Answer
Showing a trend over time
Q4 · Multiple Choice
What is Seaborn built on top of?
  • a) pandas
  • b) NumPy
  • c) matplotlib
  • d) scikit-learn
Answer
matplotlib
Q5 · Code Trace
What does this Seaborn code produce?
import seaborn as sns
sns.barplot(data=df, x='region', y='revenue')
Answer
A bar chart showing average revenue per region with confidence intervals
Q6 · Multiple Choice
Which chart type best shows the distribution of a continuous variable like salary?
  • a) Bar chart
  • b) Line chart
  • c) Pie chart
  • d) Histogram
Answer
Histogram
