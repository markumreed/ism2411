Q1 · Code Trace
What is the output?
import pandas as pd
df = pd.DataFrame({'item':['pen','book'],'price':[1.5,12.0]})
print(df['price'].mean())
Answer
6.75 — (1.5+12.0)/2 = 6.75
Q2 · Multiple Choice
What does df.head() return?
  • a) Last 5 rows
  • b) Column names only
  • c) First 5 rows
  • d) Number of rows
Answer
First 5 rows
Q3 · Multiple Choice
Which method loads a CSV into a DataFrame?
  • a) pd.read_table()
  • b) pd.load_csv()
  • c) pd.read_csv()
  • d) pd.open()
Answer
pd.read_csv()
Q4 · Code Trace
What does this code return?
df = df[df['region'] == 'South']
Answer
A DataFrame containing only rows where region equals 'South'
Q5 · Multiple Choice
Which method shows missing values per column?
  • a) df.describe()
  • b) df.isnull().sum()
  • c) df.head()
  • d) df.info()
Answer
df.isnull().sum()
Q6 · Code Trace
What does df.dropna() do?
df = df.dropna()
Answer
Returns a new DataFrame with all rows containing at least one missing value removed
