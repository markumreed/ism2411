BEFORE WEEK 1 · SETUP

Pre-Course Setup

Install Python and VS Code on your laptop before our first class meeting. This setup takes about 30 minutes and lets you write and run code from day one. If anything goes wrong, that's expected — bring questions to Week 1.

What you'll install: Python 3.12 (the language), VS Code (the editor), and the Python extension for VS Code. All free. Works on Mac, Windows, and Linux.

Step 1 — Install Python

Go to python.org/downloads and download the latest Python 3 installer for your operating system.

macOS

  1. Download the macOS 64-bit universal2 installer (.pkg file)
  2. Open the installer, click through the defaults, finish.
  3. Open Terminal (⌘+Space, type "Terminal") and run: python3 --version
  4. You should see something like Python 3.12.x

Windows

  1. Download the Windows installer (64-bit)
  2. Important: On the first screen, check the box that says "Add Python to PATH" before clicking Install Now
  3. Open PowerShell (Start menu → type "PowerShell") and run: python --version
  4. You should see Python 3.12.x
Windows note: If python --version doesn't work, you likely missed the "Add to PATH" checkbox. Reinstall and check that box.

Step 2 — Install VS Code

VS Code is the editor we'll use all semester. Get it from code.visualstudio.com.

  1. Click the big blue Download button — it auto-detects your OS
  2. Run the installer with default settings
  3. Open VS Code once it's installed

Step 3 — Install the Python Extension

  1. Open VS Code
  2. Click the Extensions icon in the left sidebar (looks like four squares) or press Ctrl+Shift+X (Windows) / ⌘+Shift+X (Mac)
  3. Search for Python — install the one published by Microsoft (it has the most downloads)
  4. While you're there, also install Pylance (also by Microsoft) — it gives smarter autocomplete

Step 4 — Verify it all works

  1. In VS Code, go to File → New File
  2. Save it as hello.py on your Desktop
  3. Type this single line and save: 
    print("Hello, ISM2411")
  4. Right-click in the editor and choose "Run Python File in Terminal"
  5. The terminal should print Hello, ISM2411
If you see Hello, ISM2411 in the terminal, you're done. You have a working Python development environment. Welcome to the course.

Step 5 — Create a DataCamp account

USF has a free DataCamp for Classrooms license for this course. You'll get an invite link from the instructor in Week 1. No setup needed before then — just be ready to accept the invite when it arrives in your USF email.

Common problems

"python is not recognized" on Windows

You forgot to check "Add Python to PATH" during install. Uninstall Python from Settings → Apps, then reinstall with that box checked.

"command not found: python" on Mac

On Mac, the command is python3, not python. Both work in VS Code once the extension is installed.

VS Code says "No Python interpreter selected"

Press Ctrl+Shift+P (or ⌘+Shift+P on Mac), type "Python: Select Interpreter", and choose the Python 3.12 entry.

Multiple Python versions installed

This is common on Mac. As long as VS Code's interpreter is set to Python 3.10+, you're fine.

Still stuck? Bring your laptop to Week 1. The first 15 minutes are reserved for setup troubleshooting.
