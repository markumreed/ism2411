Selecting columns

Filtering rows

df[df["price"] > 100]
df[df["region"] == "North"]
df[(df["price"] > 50) & (df["region"] == "South")]

df[df["product"].str.contains("Pro")]
Use & and |, not and/or. Parens around each condition.

Cleaning & Transforming

Missing data

df.dropna()               # remove rows with ANY null

df["price"].fillna(df["price"].mean())

df[mask]

Apply a function

def tier(price):
    if price > 100: return "high"
    if price > 50:  return "mid"
    return "low"

df["tier"] = df["price"].apply(tier)
# or shorter:
df["tier"] = df["price"].apply(
    lambda p: "high" if p > 100 else "low"
)

Chart Type Guide

Bar chart

Compare quantities across categories — e.g. sales by region, revenue by product.

Line chart

Show trends over time — e.g. monthly revenue, weekly orders, stock price over a year.

Histogram

Show the distribution of a single numeric variable — e.g. price spread, order size.

Scatter plot

Show the relationship between two numeric variables — e.g. price vs quantity sold.

Box plot

Show distribution and outliers across groups — e.g. salary ranges by department.

Common Errors — Unit 4
  • SettingWithCopyWarning — assign to .loc[] not directly to a filtered slice: df.loc[mask, "col"] = value.
  • KeyError when selecting a column — check df.columns for exact spelling and case; column names are case-sensitive.
  • Using and/or in boolean filters — use & and | instead, and wrap each condition in parentheses: (df["price"] > 50) & (df["region"] == "South").
  • Forgetting reset_index() after groupby — the grouped column becomes the index; call .reset_index() to make it a regular column again.
