What you'll learn

The Foundations unit teaches you what a computer actually is, how to communicate with it through the command line, and the fundamental vocabulary of Python — variables, data types, operators, and conditionals. By the end of Unit 1 you can write small scripts that accept input, make decisions, and print results. These skills are the alphabet of programming: every concept in every subsequent unit is built on what you learn here, so it is worth taking the time to understand not just what the commands do but why they work the way they do. Students who invest in Module 1 foundations consistently outperform those who skip ahead.

Unit Outcomes

After completing Unit 1, you will be able to:

  • Explain what a CPU, file system, and path are in plain English — without jargon
  • Navigate a file system using the command line (cd, ls/dir, mkdir, pwd)
  • Write Python scripts that use variables, data types (str, int, float, bool), and print()/input()
  • Apply arithmetic, comparison, and logical operators to compute business formulas
  • Write conditional logic with if/elif/else to automate simple business decisions

Module Connections

Unit 1 is the entry point to the entire course — there is no prerequisite. It connects forward to Unit 2, where the variables and conditionals you learn here become the building blocks for loops and functions. Every concept in Units 2–5 assumes fluency with the core data types and control structures covered in Modules 3–5. Students who return to the Unit 1 cheatsheet during later units find that most "new" syntax is just a combination of things they already know from these first five modules.
