What you'll learn

The Foundations unit teaches you what a computer actually is, how to talk to it through the command line, and the basic vocabulary of Python — variables, data types, operators, and conditionals. By the end of Unit 1 you can write small scripts that take input, make decisions, and print results. The skills here are the alphabet of programming. Everything that follows is built on them.

Weeks in this unit

W1
What is a Computer?
How computers work: CPU, memory, storage — in plain English, What files are, what a file system is, what a path means, Directories: how folders nest and why this matters
W2
The Command Line & Your Environment
Terminal basics: cd, ls / dir, mkdir, pwd, clear, Relative vs. absolute file paths, Installing Python and VS Code
W3
Variables, Data Types & First Output
Variables as labeled containers, Strings, integers, floats, booleans, print() and input()
W4
Operators & Expressions
Arithmetic, comparison, and logical operators, Order of operations, Business formulas: revenue = price × quantity, margin = (price − cost) / price
W5
Conditionals
if, elif, else, Writing decision logic, Nested conditionals — and when to avoid them
