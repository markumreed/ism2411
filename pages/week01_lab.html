Get fluent with the vocabulary of files, paths, and directories — no coding yet.

Exercises

EXERCISE 1 BASIC
Trace a real file path
Open Finder (Mac) or File Explorer (Windows). Pick any file in your Downloads folder. Right-click → Properties / Get Info and copy the full path. Write it down. Identify (a) the root, (b) the directory chain, (c) the filename, (d) the extension.
Expected: a written-out path like /Users/yourname/Downloads/syllabus.pdf with each component labeled
EXERCISE 2 BASIC
Map your project folder
On your machine, create a folder called ism2411 wherever you keep school work. Inside it, create one subfolder per unit (unit1, unit2, ...). We will fill these up over the semester.
Expected: folder structure visible in Finder/Explorer — ism2411/unit1/, ism2411/unit2/, etc.
EXERCISE 3 BASIC
Vocabulary check
Without looking, define in one sentence each: CPU, RAM, storage, file, directory, path, extension, program, interpreter.
Expected: 9 one-sentence definitions written in your own words — not copied from the reading
EXERCISE 4 BASIC
Discussion (in pairs or async forum)
Name three things you currently do in Excel that take more than 10 minutes. Which of those could be automated by a program that runs in a few seconds?
Expected: 3 tasks described with brief explanation of why each could or could not be automated
EXERCISE 5 INTERMEDIATE
Draw the directory tree
Given the following three file paths, draw the directory tree they imply. Show which folders exist and how they nest. Then write one relative path from one file to another.

/home/alice/projects/q1_report/data/sales.csv
/home/alice/projects/q1_report/scripts/summary.py
/home/alice/projects/q2_report/data/sales.csv
Expected: a hand-drawn or text-format tree; a relative path like ../../q2_report/data/sales.csv from the scripts folder
EXERCISE 6 INTERMEDIATE
RAM vs. Storage scenarios
For each scenario below, identify whether the problem is in RAM, storage, or something else, and explain your reasoning:

(a) You edited a spreadsheet for an hour and your laptop died. When you restart, your changes are gone.
(b) You try to open a CSV file and get an error saying the file cannot be found.
(c) Your Python script is running slowly when processing a 2 GB file.
Expected: (a) RAM — changes not saved to storage; (b) storage — file path wrong or file missing; (c) storage I/O or RAM capacity

Stretch Challenge

STRETCH A CHALLENGE
Find a .py file and explain it
Search your machine or the internet for any short Python script (5–15 lines). Open it in a text editor. Read it line by line. Write one sentence explaining what each line does — even if you are guessing. This is practice for reading unfamiliar code, which you will do constantly in your career.
STRETCH B CHALLENGE
Research: how much RAM do real workloads need?
Look up: how much RAM does Python typically use to hold a pandas DataFrame with 1 million rows and 10 columns of floats? Why does this matter for analysts working with large data sets? Write a 3–4 sentence explanation.

Reflection

Self-Assessment Questions
  1. Before this module, how would you have explained what happens when you open an Excel file? How has your answer changed?
  2. Which of the nine vocabulary terms from Exercise 3 are you least confident about? What would help you solidify that concept?
  3. The business discussion (Exercise 4) asked about tasks you could automate. What would you need to learn — beyond this module — to actually build that automation?
Submission

No code submission this module. Bring your vocabulary answers and your folder structure to class. Post your Exercise 4 automation examples to the course discussion forum before lab.
