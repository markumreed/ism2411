Unit 1 — Foundations

Variables, types, operators, conditionals, command line. Everything Python lets you talk about — before loops or functions get involved.

How to use this: Print it. Annotate it. Bring it to lab. Cheatsheets are syntax-only — they don't teach concepts. Read the unit's weekly pages first, then come back here when you forget syntax mid-task.

Variables

name = "Alex"        # str
age = 21             # int
gpa = 3.7            # float
is_senior = True     # bool

Names: lowercase, underscores. No spaces, no leading digits.

3 + 2     # 5
3 - 2     # 1

Comparison operators

x != y     # not equal

Logical operators

a and b    # both true

not a      # negate

# Truth table reminder:
# True and False  → False

Conditionals

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"
Indent with 4 spaces. Colons matter. No braces.

String methods

name = "hello world"
name.upper()          # "HELLO WORLD"
name.lower()          # "hello world"
name.capitalize()     # "Hello world"
name.strip()          # removes whitespace
name.replace("o","0") # "hell0 w0rld"
name.split(" ")       # ["hello", "world"]
" ".join(["a","b"])   # "a b"
name.startswith("h")  # True
"world" in name       # True

Number formatting

price = 1234.5678
f"{price:.2f}"        # "1234.57"
f"{price:,.2f}"       # "1,234.57"
f"{price:.0f}"        # "1235"

pct = 0.1537
f"{pct:.1%}"          # "15.4%"

Type checking

type(42)          # <class 'int'>
type("hi")        # <class 'str'>
type(3.14)        # <class 'float'>
type(True)        # <class 'bool'>

isinstance(42, int)      # True
isinstance("hi", str)    # True
isinstance(3.14, (int, float))  # True

Truthy / falsy

# Falsy (treated as False):
bool(0)      # False
bool("")     # False
bool(None)   # False
bool([])     # False
bool({})     # False

# Truthy (everything else):
bool(1)      # True
bool("x")   # True
bool([0])   # True

Common Errors — Unit 1

Watch out for these

SyntaxError — Missing colon after if, for, or def.
Python expects a : at the end of every block header. if x > 0 is wrong; if x > 0: is right.

TypeError — Cannot do "5" + 5. You cannot add a string and an integer directly. Use int("5") + 5 or "5" + str(5) depending on what you need.

IndentationError — Inconsistent indentation inside an if block. Python is whitespace-sensitive. Stick to 4 spaces; never mix tabs and spaces.

NameError — Variable used before assignment. If Python says a name is not defined, check that you spelled it correctly and that you assigned it before the line where it's used.
