Exercises

EXERCISE 1
BASIC
Load it
Run import pandas as pd; df = pd.read_csv('data/retail_sales.csv'). Then print .head(), .shape, .columns.tolist(), and .dtypes.
Expected output: shape should be (5000, 6) or similar; dtypes should show object for text columns and float64/int64 for numbers.
EXERCISE 2
BASIC
Filter by region
Create south = df[df['region'] == 'South']. Print len(south). Then print south.head() to confirm only South rows remain.
Expected output: a row count (some fraction of 5000) and a head showing 'South' in every region cell.
EXERCISE 3
BASIC
Top 5 by revenue
Run df.sort_values('revenue', ascending=False).head(5). Print the result. Note the index values — they are not 0–4 because you haven't reset the index.
Expected output: 5 rows with the highest revenue values, descending order.
EXERCISE 4
INTERMEDIATE
Two-condition filter
Create big_south = df[(df['region']=='South') & (df['revenue']>1000)]. Note the & and parentheses — both are required. Print the length and the first 5 rows.
Expected output: fewer rows than either filter alone; all rows should be South AND revenue > 1000.
EXERCISE 5
INTERMEDIATE
A new column
Create df['per_unit'] = df['revenue'] / df['quantity']. Print .head() again to confirm the new column appears. Then print the mean per-unit price across all rows.
Expected output: head() shows a new per_unit column; mean() returns a single float.
EXERCISE 6
INTERMEDIATE
Unique values
Print df['region'].unique() and df['region'].value_counts(). The first shows all distinct values; the second shows how many rows each value appears in.
Expected output: array of region names; then a count table showing North, South, East, West (with possible nulls).
EXERCISE 7
INTERMEDIATE
Select columns
Create a new DataFrame with only the columns region, product, and revenue. Print its shape. Notice that selecting a subset of columns is a quick way to reduce memory and focus your analysis.
EXERCISE 8
BASIC
Save
Run df.to_csv('data/explored.csv', index=False). Verify the file exists. Open it in a text editor to confirm the header row looks correct.

Stretch Challenge

STRETCH 1 · CHALLENGE
Top 3 products by average per-unit price in the South
Chain together: filter to South, group by product, compute mean per_unit, sort descending, take top 3. You will need concepts from Module 15 (groupby), but try to reason through it. Hint: south.groupby('product')['per_unit'].mean().sort_values(ascending=False).head(3).
STRETCH 2 · CHALLENGE
What fraction of total revenue comes from orders above $500?
Filter to big orders, sum their revenue, divide by total revenue, multiply by 100. Express your answer as a percentage rounded to 1 decimal place. Use an f-string to print it cleanly.

3. If a colleague who only knows Excel asked you to explain why you would choose pandas for this task, what is the single most compelling reason you would give?
