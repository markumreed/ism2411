Unit 5 — Capstone

Module 16 · 1 module

What you'll learn

The Capstone is the culminating project for ISM2411: a single sustained analysis of a retail sales dataset containing approximately 5,000 rows. There are no new concepts in Unit 5 — only integration. You will apply the complete analytical workflow from Modules 13–15: load the raw data, clean it with documented decisions, formulate 3–5 genuine business questions, answer them with pandas code and properly labeled visualizations, and present your most interesting finding in a 5–7 minute class presentation. This is the deliverable you will add to your portfolio and show in job interviews to demonstrate that you can do real data work with Python.

Unit Outcomes

After completing Unit 5, you will be able to:

  • Execute a complete end-to-end data analysis workflow — from raw CSV to clean data to charts to written findings
  • Formulate testable business questions and answer them with code and quantitative evidence
  • Produce professional, labeled visualizations appropriate to each question type
  • State analytical findings as numbers with business context, not vague opinions
  • Present a data finding clearly to a mixed technical/non-technical audience in under 7 minutes

Module Connections

Unit 5 draws on every prior unit in the course. The command-line and Git skills from Units 1–2 are essential for setting up your project repo and submitting via GitHub URL. The loops and functions from Unit 2 underpin the pandas operations you use throughout. The data structures from Unit 3 explain how pandas works under the hood. Modules 13–15 provide the direct technical toolkit: loading, cleaning, aggregating, and visualizing. The Module 15 lab (Aggregate & Chart — Capstone Warm-up) is the direct predecessor to this module; students who completed it thoroughly will find Module 16 to be a natural extension rather than a sudden jump in difficulty.
