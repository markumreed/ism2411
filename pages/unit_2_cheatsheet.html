Unit 2 — Control Flow & Structure

Loops, functions, debugging idioms, and Git/GitHub. The patterns that turn one-off scripts into reusable code.

How to use this: Print it. Annotate it. Bring it to lab. Cheatsheets are syntax-only — they don't teach concepts. Read the unit's weekly pages first, then come back here when you forget syntax mid-task.

for loop

for i in range(5):
    print(i)        # 0,1,2,3,4

for name in ["Alex", "Sam", "Lee"]:
    print(name)

for i in range(1, 11):    # 1..10
    print(i)

for i in range(0, 20, 2): # step 2
    print(i)

while loop

count = 0
while count < 5:
    print(count)
    count += 1     # don't forget this!

# break / continue
for n in range(10):
    if n == 3:
        continue   # skip 3
    if n == 7:
        break      # stop entirely
    print(n)

Multiple returns

def stats(nums):
    return min(nums), max(nums), sum(nums)/len(nums)

lo, hi, avg = stats([1, 2, 3, 4])

Scope

total = 0   # global

def add_one():
    total = 1    # this is LOCAL
    print(total)

# global stays at 0 unless you write:
def add_one():
    global total
    total += 1
Avoid global. Return values instead.

Debugging idioms

# 1. Print the variable
print(f"x = {x!r}, type = {type(x)}")

# 2. Print at every step
for i, item in enumerate(items):
    print(i, item)

# 3. Use assert to catch wrong assumptions
assert len(items) > 0, "items is empty!"

enumerate() and zip()

items = ["a", "b", "c"]

# enumerate: get index + value
for i, item in enumerate(items):
    print(i, item)   # 0 a, 1 b, 2 c

prices = [1.0, 2.0, 3.0]
# zip: pair two lists
for item, price in zip(items, prices):
    print(item, price)

List comprehensions

nums = [1, 2, 3, 4, 5]

# transform
squares = [n**2 for n in nums]

# filter
evens = [n for n in nums if n % 2 == 0]

# transform + filter
big_sq = [n**2 for n in nums if n > 2]

# business: apply 10% discount to all prices
discounted = [p * 0.9 for p in prices]

Default arguments & *args

def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

greet("Alex")           # Hello, Alex!
greet("Alex", "Hi")     # Hi, Alex!

def total(*prices):     # any number of args
    return sum(prices)

total(1, 2, 3, 4)       # 10

Common Errors — Unit 2

Watch out for these

Off-by-one errors in range() — range(1, 5) produces 1, 2, 3, 4 — it does not include 5. The second argument is always exclusive. If you need 1 through 5, write range(1, 6).

Infinite while loop — Always ensure the loop condition can eventually become False. If you forget to update the variable being tested (e.g., count += 1), the loop runs forever. Use Ctrl+C to stop a runaway script.

Forgetting return — A function without a return statement silently returns None. If your variable is unexpectedly None, check that your function actually returns the computed value.

Modifying a list while iterating over it — Removing or adding items inside a for loop that iterates the same list causes skipped or duplicated items. Iterate over a copy instead: for item in items[:]:
