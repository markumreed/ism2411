Today's walkthrough

Variables are labeled containers

A variable is a name attached to a value. You assign with = (single equals). The name is on the left, the value is on the right.

product = "Notebook"
price = 4.99
quantity = 12

The four types you will use most

Python has many types. These four cover almost everything you will do in this course.

name = "Alice"      # str  — text in quotes
age = 30             # int  — whole number
price = 19.99        # float — decimal number
in_stock = True      # bool — True or False (no quotes)

print() and f-strings

print() writes to your terminal. An f-string is a string with an f in front — variables in curly braces get substituted in.

product = "Notebook"
price = 4.99
quantity = 12
total = price * quantity
print(f"{quantity} x {product} = ${total}")
# 12 x Notebook = $59.88

input() for user input

input() pauses the program and waits for the user to type something and press Enter. It always returns a string — convert with int() or float() if you need a number.

name = input("Your name: ")
print(f"Hello, {name}!")

Jupyter vs. VS Code

A .py file in VS Code is one big script that runs top to bottom. A Jupyter notebook is a series of cells you can run one at a time, with output (including charts) right next to the code. We use VS Code for scripts and Jupyter for exploration — you will see both this semester.

In-class activities

  • What is the type of "12" vs. 12? Try it in the terminal: type("12") and type(12).
