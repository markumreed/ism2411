Exercises

EXERCISE 1
BASIC
Q1: Revenue by region
Load clean_sales.csv. Compute total revenue per region using groupby("region")["revenue"].sum().reset_index(). Sort descending. Render as a bar chart with a title, axis labels, and appropriate colors. Save as charts/q1_revenue_by_region.png.
Required: title, x-label, y-label, saved PNG. Write one sentence below your chart code stating the top region and its dollar amount.
EXERCISE 2
INTERMEDIATE
Q2: Monthly revenue trend
Create df['month'] = df['order_date'].dt.month (parse the date first if needed). Group by month, sum revenue, render as a line chart with markers. Save as charts/q2_monthly_trend.png.
Required: markers on data points, all 12 months on x-axis, title and labels. Write one sentence stating the peak month and percentage above average.
EXERCISE 3
BASIC
Q3: Order size distribution
Plot a histogram of df['revenue'] with bins=20. Save as charts/q3_order_distribution.png. Print the median and mean order size. Write one sentence stating whether the distribution is right-skewed, left-skewed, or approximately symmetric, and what that means for a business analyst choosing between mean and median.
EXERCISE 4
INTERMEDIATE
Q4: Your own business question
Add a 4th chart that answers a question you find genuinely interesting. Justify your chart type choice in a comment. Save as charts/q4_custom.png. This should be something a real analyst or manager would want to know.
Graded on: chart type appropriateness, correct labeling, and quality of the business question.
EXERCISE 5
INTERMEDIATE
Seaborn version
Replace one of your matplotlib charts (Q1, Q2, or Q3) with a Seaborn equivalent (sns.barplot, sns.lineplot, or sns.histplot). In a comment, describe one difference you observe — aesthetically, informationally, or in the amount of code required.
EXERCISE 6
BASIC
Multiple aggregations
Use .agg() to compute both total revenue and order count per region in one call. Print the resulting DataFrame. Which region has the most orders? Which has the highest average order value (total revenue / order count)? Answer in a comment.
EXERCISE 7
BASIC
Notebook tidy-up
If working in Jupyter, add a markdown cell before each chart block with a level-2 heading (##) and a one-sentence business question the chart answers. Commit the notebook. If working in a .py file, add a section comment block before each chart.

Stretch Challenge

STRETCH 1 · CHALLENGE
Side-by-side subplots
Use plt.subplots(1, 2, figsize=(14,5)) to create a figure with two side-by-side charts: revenue by region on the left and revenue by product category on the right. Share a y-axis label. Save as a single PNG. This is the kind of layout used in executive dashboards.
STRETCH 2 · CHALLENGE
Regional monthly heatmap
Pivot the data: rows = region, columns = month, values = total revenue. Use df.pivot_table() and then sns.heatmap() with annot=True, fmt=".0f". This shows which region/month combinations drove the most revenue at a glance.

Reflection

Self-assessment

1. Look at each of your four charts. If someone who had never seen this dataset looked at them for 10 seconds, could they state the key finding? If not, what needs to change — the chart type, the labels, or the title?

2. The capstone requires you to state your findings as numbers with context: "Region X drove $Y in revenue, which is Z% above the average." Practice writing that sentence for each of your three required charts.

3. This lab is the capstone dress rehearsal. What felt difficult or unclear? Note it now — the capstone is a longer version of exactly this workflow, and Module 16 gives you dedicated time to finish.
