Course Glossary

Every key term introduced across all 16 modules, organized alphabetically. Each entry shows which unit introduced the concept and includes a code example where applicable.

A
Accumulator patternModule 06 · Loops
A variable initialized before a loop that collects or builds up a result with each iteration. The most common accumulators are running totals (sum) and growing lists (append).
total = 0
for sale in [100, 250, 75]:
    total += sale   # accumulates to 425
ArgumentModule 07 · Functions
The actual value passed to a function when it is called. Contrast with parameter, which is the placeholder name in the function definition.
def greet(name):   # name is the parameter
    return f"Hello, {name}"

greet("Alice")     # "Alice" is the argument
See also: Parameter, Return value
AggregationModule 15 · Grouping
A calculation that reduces many values to one summary value — sum, mean, count, min, max. In pandas, aggregations are applied to groups created by groupby().
df.groupby("region")["sales"].sum()
df.groupby("region")["sales"].agg(["sum","mean","count"])
See also: groupby()
Arithmetic operatorModule 04 · Operators
A symbol that performs a mathematical operation: + (add), - (subtract), * (multiply), / (divide), // (integer divide), % (modulo/remainder), ** (exponent).
7 / 2    # 3.5  — float division
7 // 2   # 3    — integer division (floor)
7 % 2    # 1    — remainder
B
BooleanModule 03 · Variables
A data type with exactly two values: True or False. The result of any comparison expression is a boolean. Named after mathematician George Boole.
is_active = True
price > 100       # True or False
type(True)        # <class 'bool'>
See also: Truthy / Falsy
breakModule 06 · Loops
A statement that immediately exits the nearest enclosing loop, skipping any remaining iterations. Use it when you've found what you were looking for and don't need to continue.
for item in inventory:
    if item["stock"] == 0:
        print("Out of stock found!")
        break   # stop checking once we find one
See also: continue
C
Command line (CLI)Module 02 · Terminal
A text-based interface for talking to your operating system by typing commands. Also called the terminal, shell, or command prompt. Python scripts are run from the command line with python filename.py.
pwd        # print working directory
ls         # list files (mac/linux)
cd folder  # change into a folder
python script.py  # run a Python file
Comparison operatorModule 04 · Operators
An operator that compares two values and returns a boolean. The six comparison operators are: == (equal), != (not equal), <, <=, >, >=.
5 == 5    # True
5 != 3    # True
"a" < "b" # True  — strings compare alphabetically
See also: Boolean, Logical operator
CapstoneModule 16 · Capstone
The final project of the course. Students load a real retail dataset, clean it, answer three or more business questions with code, produce at least three charts, push the finished notebook to GitHub, and present their findings to the class in 5–7 minutes.
See also: Capstone Rubric, Capstone Guide
continueModule 06 · Loops
A statement that skips the rest of the current loop iteration and jumps to the next one. The loop does not exit — it continues with the next value.
for n in range(10):
    if n % 2 == 0:
        continue    # skip even numbers
    print(n)        # prints only 1,3,5,7,9
See also: break
CPU (Central Processing Unit)Module 01 · Computer
The chip inside your computer that executes instructions one at a time, billions of times per second. When you run a Python script, the interpreter translates your code into instructions the CPU can execute.
See also: RAM, Storage
csv moduleModule 12 · Files
Python's built-in module for reading and writing CSV (comma-separated values) files. csv.reader() iterates row by row; csv.DictReader() gives each row as a dictionary keyed by the header.
import csv
with open("sales.csv") as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(row["product"], row["revenue"])
See also: DataFrame, File handle
D
DataFrameModule 13 · pandas
The primary data structure in pandas. A two-dimensional table with labeled rows (index) and labeled columns — like a spreadsheet in Python. Created with pd.read_csv(), pd.DataFrame(), or other pandas constructors.
import pandas as pd
df = pd.read_csv("sales.csv")
df.shape       # (5000, 6) — rows, columns
df.columns     # Index(['date','product','sales'...])
See also: Series
Data typeModule 03 · Variables
The category of a value that determines what operations are valid on it. Python's core built-in types are int, float, str, bool, NoneType, list, dict, and tuple.
type(42)       # <class 'int'>
type(3.14)     # <class 'float'>
type("hello")  # <class 'str'>
type(True)     # <class 'bool'>
defModule 07 · Functions
The keyword used to define a function in Python. Everything indented under the def line is the function body and runs whenever the function is called.
def calculate_margin(price, cost):
    return (price - cost) / price
See also: Function, Return value
DictionaryModule 11 · Dictionaries
A collection of key-value pairs. Keys are unique identifiers (usually strings); values can be any type. Dictionaries give O(1) lookup — finding a value by key is instant regardless of size. Written with curly braces {}.
customer = {"name": "Alice", "balance": 1500.0}
customer["name"]                    # "Alice"
customer.get("email", "unknown")    # safe access
See also: Key-value pair, List
DirectoryModule 01 · Computer
A container in the file system that holds files and other directories. Also called a folder. The arrangement of directories forms a tree structure, starting from the root (/ on Mac/Linux, C:\ on Windows).
See also: Path, File
E
ExpressionModule 04 · Operators
Any combination of values, variables, and operators that Python evaluates to produce a single result. Expressions can be as simple as 5 or as complex as (price * qty) * (1 - discount).
revenue = price * quantity     # price * quantity is an expression
is_profitable = margin > 0.2   # comparison expression
explain() / describe()Module 14 · Cleaning
df.describe() returns summary statistics for all numeric columns: count, mean, standard deviation, minimum, 25th percentile (Q1), median (50%), 75th percentile (Q3), and maximum. A fast first look at numeric data quality.
df.describe()
df.describe(include="object")   # for string columns
df["price"].describe()
F
f-stringModule 03 · Variables
A string literal prefixed with f that allows embedding Python expressions directly inside {} placeholders. The cleanest way to build strings with variable values in Python 3.6+.
name = "Alice"
total = 1234.5
print(f"Hello, {name}! Your balance is ${total:,.2f}")
FileModule 01 · Computer
A named collection of data stored on disk. Files have a name, an extension (like .py, .csv, .txt) that indicates the format, and a location in the file system described by their path.
See also: Directory, Path
File handleModule 12 · Files
An object returned by open() that represents an open connection to a file on disk. You read or write through the handle. Always close it when done — the with statement does this automatically.
with open("report.txt", "r") as f:
    content = f.read()
# f is automatically closed after the with block
See also: with statement
FloatModule 03 · Variables
Python's data type for decimal numbers (floating-point). Floats can represent very large and very small values but may have small rounding errors due to how binary computers store decimals.
price = 19.99
0.1 + 0.2   # 0.30000000000000004  ← rounding error
round(0.1 + 0.2, 2)   # 0.3
See also: Integer
for loopModule 06 · Loops
A loop that iterates over each item in an iterable (list, string, range, dictionary, etc.) and executes the loop body once per item. The variable after for takes on each value in turn.
for product in ["Widget", "Gadget", "Doohickey"]:
    print(product.upper())

for i in range(1, 6):   # 1, 2, 3, 4, 5
    print(i)
See also: while loop, range()
FunctionModule 07 · Functions
A named, reusable block of code that performs a specific task. Defined with def, takes zero or more parameters, and optionally returns a value. Functions reduce repetition and make code easier to test and reason about.
def calculate_tax(price, rate=0.07):
    return price * rate

tax = calculate_tax(100)     # 7.0
tax = calculate_tax(100, 0.10)  # 10.0
See also: Parameter, Return value, Scope
G
GitModule 08 · Git
A distributed version control system that tracks changes to files over time. Git stores snapshots (commits) of your project so you can see what changed, when, and why — and revert to any previous state.
git init              # start tracking
git add analysis.py   # stage a file
git commit -m "Add revenue calculation"  # save snapshot
git log               # view history
See also: GitHub, Commit
GitHubModule 08 · Git
A web platform that hosts Git repositories in the cloud. Used in this course to submit all work from Module 8 onward. A public GitHub profile with real analysis projects is a portfolio asset for job applications.
git remote add origin https://github.com/you/repo
git push -u origin main   # first push
git push                  # subsequent pushes
groupby()Module 15 · Grouping
A pandas method that splits a DataFrame into groups based on the unique values of one or more columns, then applies an aggregation function to each group. Equivalent to SQL's GROUP BY.
# Total sales per region
df.groupby("region")["sales"].sum()

# Multiple aggregations
df.groupby("region")["sales"].agg(["sum","mean","count"])
See also: Aggregation
H
head() / tail()Module 13 · pandas
df.head(n) returns the first n rows of a DataFrame (default 5). df.tail(n) returns the last n rows. Essential for quickly inspecting a dataset after loading it.
df.head()      # first 5 rows
df.head(10)    # first 10 rows
df.tail(3)     # last 3 rows
I
if / elif / elseModule 05 · Conditionals
Python's conditional branching syntax. if tests the first condition; elif (else-if) tests additional conditions in order; else runs if none of the conditions above were true. Only one branch executes.
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"
ImmutableModule 10 · Lists & Tuples
A value that cannot be changed after it is created. Strings, integers, floats, booleans, and tuples are immutable. Contrast with mutable types like lists and dictionaries.
t = (1, 2, 3)
t[0] = 99     # TypeError: tuple does not support item assignment
# but the variable can be reassigned:
t = (99, 2, 3)  # this creates a new tuple
See also: Mutable, Tuple
Index (list)Module 10 · Lists & Tuples
A number specifying the position of an element in a list or tuple. Python uses zero-based indexing: the first element is at index 0, the last at index -1. Accessing an index that doesn't exist raises IndexError.
items = ["a", "b", "c", "d"]
items[0]    # "a"  — first
items[-1]   # "d"  — last
items[1:3]  # ["b","c"]  — slice
IntegerModule 03 · Variables
Python's data type for whole numbers (no decimal point). Integers can be arbitrarily large in Python — there's no overflow. Division of two integers with / always returns a float; use // for integer (floor) division.
quantity = 42
10 / 3    # 3.3333...  (float)
10 // 3   # 3          (integer)
InterpreterModule 01 · Computer
A program that reads and executes source code line by line, translating it into machine instructions as it goes. Python is an interpreted language — you don't compile it before running it, which makes development faster and errors easier to see.
python script.py   # interpreter reads and runs script.py
See also: Program
IterableModule 06 · Loops
Any object that can be iterated over (looped through) one item at a time. Strings, lists, tuples, dictionaries, ranges, and file objects are all iterables. A for loop works on anything iterable.
for char in "hello":    # string is iterable
    print(char)

for key in {"a":1,"b":2}:  # dict keys are iterable
    print(key)
J
Jupyter NotebookModule 03 · Variables
An interactive computing environment where code, output, and prose live in the same document. Notebooks consist of cells — a code cell runs Python; a markdown cell renders formatted text. The standard format for data analysis work. Files end in .ipynb.
# In a code cell:
import pandas as pd
df = pd.read_csv("data.csv")
df.head()   # output appears directly below
K
Key-value pairModule 11 · Dictionaries
The fundamental unit of a dictionary. A key is an immutable identifier (usually a string) used to look up the associated value, which can be any Python object. Accessing a key that doesn't exist raises KeyError — use .get() to avoid this.
d = {"product": "Widget", "price": 9.99}
d["product"]            # "Widget"
d.get("color", "N/A")  # "N/A" instead of KeyError
See also: Dictionary
L
ListModule 10 · Lists & Tuples
An ordered, mutable collection of items. Items can be of mixed types. Lists are defined with square brackets [] and support indexing, slicing, appending, removing, and sorting. The most commonly used collection type in Python.
prices = [9.99, 14.99, 4.50, 22.00]
prices.append(7.99)  # add to end
prices.sort()        # sort in place
prices[0]            # 4.50 (after sort)
See also: Tuple, Dictionary
Logical operatorModule 04 · Operators
An operator that combines boolean values: and (both must be True), or (at least one must be True), not (inverts True/False). Used to build compound conditions in if statements and pandas filters.
if age >= 18 and has_id:
    print("Can enter")

# In pandas: use & and | (not and/or)
df[(df["price"] > 10) & (df["region"] == "South")]
LoopModule 06 · Loops
A control structure that repeats a block of code multiple times. Python has two loop types: for (iterates over a sequence) and while (repeats as long as a condition is True). Loops are how you apply an operation to every item in a dataset.
See also: for loop, while loop
M
MeanModule 14 · Cleaning
The average of a set of numbers: sum divided by count. Sensitive to outliers — a few very large or small values pull it away from the typical value. Compare to median for skewed data.
df["sales"].mean()
import statistics; statistics.mean([10, 20, 30])  # 20.0
See also: Median
MedianModule 14 · Cleaning
The middle value when a dataset is sorted in order. Half the values fall above it, half below. More robust than the mean when the data has outliers — use median for income, house prices, or anything skewed.
df["sales"].median()
See also: Mean
Missing value (NaN)Module 14 · Cleaning
A cell in a DataFrame with no value. Represented as NaN (Not a Number) in pandas, inherited from NumPy. Missing values propagate through calculations — adding NaN to anything gives NaN. Must be handled before analysis.
df.isnull().sum()              # count per column
df.dropna()                    # remove rows with any NaN
df["price"].fillna(df["price"].mean())  # replace with mean
Module (Python)Module 07 · Functions
A Python file that contains functions, classes, and variables that can be imported into other scripts. The standard library has hundreds of built-in modules (csv, math, os); third-party modules like pandas and matplotlib are installed with pip.
import csv
import pandas as pd
from math import sqrt, pi
MutableModule 10 · Lists & Tuples
A value that can be changed in place after it is created. Lists and dictionaries are mutable. Mutability matters when passing objects to functions — the function can modify the original list, which can cause bugs if unexpected.
prices = [10, 20, 30]
prices[0] = 99       # mutates in place
prices.append(40)    # mutates in place
print(prices)        # [99, 20, 30, 40]
See also: Immutable
N
None / NoneTypeModule 03 · Variables
Python's null value — the absence of a value. A function that doesn't have a return statement returns None automatically. Often used as a default for optional parameters. Check for it with is None, not == None.
def greet(name):
    print(f"Hello, {name}")   # no return!

result = greet("Alice")   # prints, but...
print(result)             # None
O
OperatorModule 04 · Operators
A symbol that performs an operation on one or more values (operands). Python has arithmetic, comparison, logical, assignment, and membership operators. Operators have precedence (order of operations) — parentheses override.
See also: Arithmetic operator, Comparison operator, Logical operator
Order of operationsModule 04 · Operators
The rules determining which parts of an expression are evaluated first. Python follows PEMDAS: Parentheses → Exponents → Multiplication/Division → Addition/Subtraction. When in doubt, use parentheses to make intent explicit.
2 + 3 * 4    # 14  (multiplication first)
(2 + 3) * 4  # 20  (parentheses override)
P
pandasModule 13 · pandas
An open-source Python library for data manipulation and analysis. Provides the DataFrame and Series data structures, plus methods for loading, cleaning, transforming, grouping, and visualizing tabular data. The de facto standard tool for business analytics in Python.
import pandas as pd
df = pd.read_csv("sales.csv")
df.groupby("region")["revenue"].sum()
ParameterModule 07 · Functions
A variable in a function definition that acts as a placeholder for values to be passed in when the function is called. Parameters have local scope — they only exist inside the function. Contrast with argument, which is the actual value passed.
def discount(price, rate):   # price and rate are parameters
    return price * (1 - rate)

discount(100, 0.10)  # 100 and 0.10 are arguments
PathModule 01 · Computer
The address of a file or directory in the file system. An absolute path starts from the root and works from anywhere (/home/user/data/sales.csv). A relative path is relative to the current working directory (data/sales.csv). Use relative paths in code you share — absolute paths break on other machines.
pd.read_csv("data/sales.csv")    # relative ✓
pd.read_csv("/home/alex/data/sales.csv")  # absolute ✗
print()Module 03 · Variables
A built-in function that displays values to standard output (the terminal or notebook). Accepts multiple arguments separated by commas. Indispensable for debugging — printing variables at each step is the simplest way to understand what your code is doing.
print("Hello")
print("Price:", price, "Total:", total)
print(f"Debug: x={x!r}, type={type(x).__name__}")
ProgramModule 01 · Computer
A sequence of instructions written in a programming language that a computer can execute. A Python program is a .py file containing Python statements. When you run it, the interpreter reads each line and passes instructions to the CPU.
R
RAM (Random Access Memory)Module 01 · Computer
The computer's working memory — where active data and running programs live. RAM is fast but temporary: it is cleared when the power is off. A pandas DataFrame loaded from a CSV lives in RAM. If a dataset is too large to fit in RAM, you need streaming or chunking techniques.
See also: CPU, Storage
range()Module 06 · Loops
A built-in function that generates a sequence of integers. range(stop) generates 0 to stop-1; range(start, stop) includes start, excludes stop; range(start, stop, step) increments by step. Commonly used with for loops.
range(5)         # 0, 1, 2, 3, 4
range(1, 6)      # 1, 2, 3, 4, 5
range(0, 20, 5)  # 0, 5, 10, 15
list(range(3))   # [0, 1, 2]
Return valueModule 07 · Functions
The value a function sends back to its caller via the return statement. Once return is reached, the function exits immediately. A function without a return statement returns None.
def square(n):
    return n * n          # sends n*n back to caller

result = square(5)        # result = 25
print(square(5) + 1)      # 26 — return value used inline
Repository (repo)Module 08 · Git
A directory tracked by Git, containing your project files and the complete history of all commits. A local repo lives on your machine; a remote repo (on GitHub) is the cloud copy. Pushing syncs local to remote.
git init          # turn current directory into a repo
git clone URL     # copy a remote repo to your machine
Rubber duck debuggingModule 07 · Functions
Explaining your code to an inanimate object — out loud, one line at a time — to find a bug. Articulating each line exposes the gap between what you think the code does and what it actually does; the bug often announces itself mid-sentence. Named in The Pragmatic Programmer (Hunt & Thomas, Addison-Wesley, 1999), after a developer who debugged by explaining his code to a small yellow rubber duck. If the duck doesn't crack it, the rehearsed explanation becomes a high-quality question for a classmate, the instructor, or an AI in Explainer mode.
See also: Traceback, print()
S
ScopeModule 07 · Functions
The region of a program where a variable is accessible. Variables defined inside a function are local (only accessible inside that function). Variables defined outside any function are global. Avoid using global — prefer returning values instead.
x = 10   # global

def double():
    x = 99   # local x — does NOT change the global x
    return x

print(double())  # 99
print(x)         # 10  — global unchanged
SeriesModule 13 · pandas
A one-dimensional labeled array in pandas — a single column of a DataFrame, or a standalone sequence with an index. Most DataFrame operations return or accept Series. Series support vectorized operations (applied to all values at once).
s = df["price"]           # Series
s.mean()                  # mean of the column
s[s > 100]                # filter the Series
df["total"] = df["price"] * df["qty"]  # Series * Series
SliceModule 10 · Lists & Tuples
A subset of a list (or string) extracted using the [start:stop:step] syntax. Start is inclusive, stop is exclusive. Slicing never raises an error for out-of-bounds indices — it just returns as much as exists.
items = [0, 1, 2, 3, 4, 5]
items[1:4]    # [1, 2, 3]
items[:3]     # [0, 1, 2]
items[::2]    # [0, 2, 4] — every other
items[::-1]   # [5, 4, 3, 2, 1, 0] — reversed
Storage (disk)Module 01 · Computer
Persistent memory — hard drives and SSDs — where files live when the power is off. Reading from or writing to disk is much slower than RAM. Python's open() function, pd.read_csv(), and df.to_csv() are all disk I/O operations.
See also: RAM, CPU
StringModule 03 · Variables
Python's data type for text. Defined with single quotes 'hello', double quotes "hello", or triple quotes for multi-line strings. Strings are immutable — methods like .upper() return a new string, they don't modify the original.
name = "Alice"
name.upper()           # "ALICE"
name[0]                # "A"
len(name)              # 5
f"Hi, {name}!"        # "Hi, Alice!"
T
TracebackModule 07 · Functions
Python's error report when a program crashes. Shows the sequence of function calls that led to the error (the "call stack"), with the most recent call last, followed by the error type and message. Read it bottom-up: the error type is on the last line, the location is one line above it.
Traceback (most recent call last):
  File "main.py", line 14, in <module>
    total = total + price
TypeError: unsupported operand type(s) for +:
           'int' and 'str'
# → price is a string; wrap it: int(price)
Truthy / FalsyModule 05 · Conditionals
Python evaluates any value as True or False in a boolean context. Falsy values are: 0, 0.0, "" (empty string), [] (empty list), {} (empty dict), None. Everything else is truthy. Used in conditions and while loops.
if items:          # True if list is non-empty
    process(items)

if not balance:    # True if balance is 0 or None
    print("No balance")
TupleModule 10 · Lists & Tuples
An ordered, immutable sequence. Defined with parentheses () or just commas. Use tuples for data that should not change — coordinates, RGB colors, database records. Functions that return multiple values return them as a tuple.
point = (40.71, -74.00)   # lat, lon — should not change
x, y = point               # unpack into variables

def min_max(data):
    return min(data), max(data)   # returns a tuple

lo, hi = min_max([3, 1, 4, 1, 5])  # lo=1, hi=5
See also: List, Immutable
Type conversionModule 04 · Operators
Explicitly changing a value from one type to another using built-in functions: int(), float(), str(), bool(). Required whenever input() returns a string that you need to treat as a number. int("hello") raises ValueError.
age = int(input("Enter age: "))   # input always returns str
price = float("19.99")
display = str(42)
int("42")    # 42  ✓
int("42.5")  # ValueError ✗ — use int(float("42.5"))
U
UTF-8Module 12 · Files
The most common text encoding standard, capable of representing every character in every human language. Always specify encoding="utf-8" when opening text files in Python to avoid encoding errors on files with accented characters, currency symbols, or emoji.
with open("data.txt", "r", encoding="utf-8") as f:
    text = f.read()
V
VariableModule 03 · Variables
A named reference to a value stored in memory. In Python, variables are created by assignment (=) and can hold any type. Python variables are dynamically typed — a variable's type is determined by the value assigned to it, and can change.
product_name = "Widget A"   # str
unit_price   = 9.99          # float
quantity     = 100           # int
is_active    = True          # bool
Version controlModule 08 · Git
A system for tracking and managing changes to files over time. Git is the standard version control system. Benefits: revert to any previous state, see exactly what changed and when, collaborate without overwriting each other's work, and provide an audit trail for data analysis projects.
See also: Git, GitHub
W
while loopModule 06 · Loops
A loop that continues executing as long as its condition evaluates to True. Unlike a for loop, the number of iterations is not known in advance. You must ensure the condition can eventually become False — otherwise you create an infinite loop.
attempts = 0
while attempts < 3:
    password = input("Password: ")
    if password == "secret":
        break
    attempts += 1   # MUST change the condition variable!
See also: for loop, break
with statementModule 12 · Files
A context manager that ensures cleanup code runs regardless of whether an error occurs. Most commonly used with open() to guarantee a file is closed when the block exits, even if an exception is raised inside it.
with open("sales.csv", "r") as f:
    data = f.read()
# f.close() is called automatically here
