Unit 3 — Data Structures

Lists, tuples, dictionaries, and reading files. The shapes Python uses to hold data — and the gateway to analytics.

How to use this: Print it. Annotate it. Bring it to lab. Cheatsheets are syntax-only — they don't teach concepts. Read the unit's weekly pages first, then come back here when you forget syntax mid-task.

Core Syntax

Lists

prices = [12.99, 5.50, 8.75]
prices[0]              # 12.99 (first)
prices[-1]             # 8.75  (last)
len(prices)            # 3
prices.append(20.00)   # add to end
prices.remove(5.50)    # remove value
prices[1] = 6.00       # change index 1
prices.sort()          # in place
sorted(prices)         # new list

Slicing

nums = [0, 1, 2, 3, 4, 5]
nums[1:4]      # [1, 2, 3]
nums[:3]       # [0, 1, 2]
nums[3:]       # [3, 4, 5]
nums[-2:]      # [4, 5]
nums[::2]      # every other → [0,2,4]
nums[::-1]     # reversed

List comprehensions

nums = [1, 2, 3, 4, 5]

# square each
[n*n for n in nums]     # [1,4,9,16,25]

# filter
[n for n in nums if n > 2]   # [3,4,5]

# transform + filter
[n*10 for n in nums if n < 4]

Tuples

point = (3, 5)     # immutable
x, y = point       # unpack
x, y = y, x        # swap!

# common as return values
def stats(d):
    return min(d), max(d)
lo, hi = stats([2, 8, 1])

Dictionaries

product = {
    "name": "Widget",
    "price": 12.99,
    "stock": 50
}
product["price"]            # 12.99
product["stock"] = 49       # update
product["sku"] = "W-001"    # add new
"price" in product          # True
del product["sku"]          # remove

Iterating dicts

for key in product:
    print(key, product[key])

for key, value in product.items():
    print(key, "=", value)

product.keys()    # all keys
product.values()  # all values

.get() — safer access

product["color"]              # KeyError!
product.get("color")          # None
product.get("color", "n/a")   # "n/a"

# good for missing keys in survey data

Reading text files

with open("data.txt") as f:
    text = f.read()       # whole file
    # or
    lines = f.readlines() # list of lines
    # or
    for line in f:
        print(line.strip())
with auto-closes the file.

Reading CSV

import csv

with open("sales.csv") as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(row["product"], row["price"])
# DictReader: each row is a dict
# keyed by the header row

Writing CSV

import csv
rows = [
    {"name": "A", "qty": 5},
    {"name": "B", "qty": 3},
]
with open("out.csv", "w", newline="") as f:
    w = csv.DictWriter(f, fieldnames=["name","qty"])
    w.writeheader()
    w.writerows(rows)

List Methods & Built-ins

List methods reference

items = [3, 1, 4, 1, 5, 9]
items.append(2)       # add to end
items.insert(0, 0)    # insert at index
items.pop()           # remove & return last
items.pop(2)          # remove & return index 2
items.remove(1)       # remove first 1 found
items.sort()          # sort in place (ascending)
items.sort(reverse=True)
items.reverse()       # reverse in place
items.count(1)        # how many 1s
items.index(5)        # first index of 5
items.copy()          # shallow copy
items.clear()         # remove all elements

Useful built-ins with lists

nums = [3, 1, 4, 1, 5, 9]
sum(nums)          # 23
min(nums)          # 1
max(nums)          # 9
sorted(nums)       # new sorted list
len(nums)          # 6
list(range(5))     # [0,1,2,3,4]
list(reversed(nums))       # reversed copy
any(n > 8 for n in nums)  # True
all(n > 0 for n in nums)  # True

Nested Data & Dict Comprehensions

Nested data structures

# list of dicts (very common!)
orders = [
    {"id": 1, "item": "Widget", "qty": 3},
    {"id": 2, "item": "Gadget", "qty": 1},
]
orders[0]["item"]       # "Widget"

# dict of dicts
catalog = {
    "W001": {"name": "Widget", "price": 9.99},
    "G001": {"name": "Gadget", "price": 24.99},
}
catalog["W001"]["price"]   # 9.99

Dict comprehensions

# from two lists
keys = ["a", "b", "c"]
vals = [1, 2, 3]
d = {k: v for k, v in zip(keys, vals)}

# transform values
prices = {"Widget": 10, "Gadget": 25}
discounted = {k: v*0.9 for k,v in prices.items()}

# filter
affordable = {k:v for k,v in prices.items() if v<20}

File Handling

File modes reference

# mode = what you can do
open(f, "r")    # read (default)
open(f, "w")    # write (overwrites!)
open(f, "a")    # append (adds to end)
open(f, "r+")   # read + write
open(f, "x")    # create new (error if exists)

# encoding: always specify for text files
open(f, "r", encoding="utf-8")

Common Errors — Unit 3
  • IndexError: list index out of range — check len() before accessing an index, or use a loop instead of hardcoded positions.
  • KeyError: key not in dict — use .get(key) or .get(key, default) instead of dict[key] when the key might be missing.
  • FileNotFoundError — check you're in the right directory (pwd in the terminal) and using relative paths correctly.
  • Mutating a list while iterating over it — iterate over a copy instead: for item in items[:]:
