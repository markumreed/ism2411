Sales Loop — Sum, Average, Max

Process a list of sales using for loops and the initialize-loop-update pattern.

Module 06 · Lab Unit 2

Exercises

EXERCISE 1
BASIC
Sum a list
Given sales = [120, 80, 250, 175, 90, 410, 60, 215], compute the total without using sum(). Use a for loop and the accumulator pattern.

STARTER CODE

sales = [120, 80, 250, 175, 90, 410, 60, 215]
total = 0       # initialize
for sale in sales:
    total += sale   # update
print(f"Total: ${total}")

EXPECTED OUTPUT

Total: $1400
EXERCISE 2
BASIC
Average
Extend Exercise 1. Track both a total and a count inside the loop, then compute the average after the loop ends. Do not use len() or sum().

EXPECTED OUTPUT

Total: $1400
Count: 8
Average: $175.00
EXERCISE 3
BASIC
Max
Find the largest sale without using max(). Track a current_max variable. Before the loop, initialize it to 0 (or to the first element). Inside the loop, compare and update.

EXPECTED OUTPUT

Largest sale: $410

INTERMEDIATE
Filtered count
Count how many sales were over $100. Use a counter variable and an if statement inside the loop. Also compute their total.

break / continue

$175

Stretch Challenge

STRETCH 1 — CHALLENGE
Running total report
Print a running total after each transaction — like a bank statement. For each sale, print the sale amount and the cumulative total so far. At the end, print how many transactions brought the running total above $500 for the first time.

Nested loops — multiplication table
