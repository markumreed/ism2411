What you'll learn

Data Structures teaches you how to organize collections of information inside a Python program and how to read and write persistent data in files. Lists for ordered sequences of items, tuples for fixed pairs of related values, dictionaries for fast key-value lookups, and the csv module for reading and writing tabular data. This unit is the bridge from "writing Python" to "doing analytics" — every dataset you will touch in Unit 4 is, underneath the pandas interface, some combination of these structures. Understanding lists and dicts makes pandas behavior predictable rather than mysterious.

Unit Outcomes

After completing Unit 3, you will be able to:

  • Create, index, slice, and modify lists; use common list methods (append, remove, sort)
  • Distinguish when to use a tuple vs. a list and apply both correctly
  • Build and query dictionaries with key-value pairs, including nested structures
  • Read and write CSV files using Python's built-in csv module
  • Use relative file paths correctly within a project repository structure

Module Connections

Unit 3 builds directly on the loops and functions from Unit 2 — iterating over a list or reading a CSV file row by row requires the loop syntax covered in Module 6. The file path concepts from Module 2 return here in Module 12 when working with file I/O. Unit 3 leads directly into Unit 4: when you call pd.read_csv() in Module 13, pandas is performing a sophisticated version of exactly what Module 12 taught you to do manually. Knowing both the manual approach and the pandas approach gives you a much deeper understanding of what the library is doing and why it behaves the way it does.
