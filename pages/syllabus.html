ISM2411 — Python for Business · USF Muma College of Business · 16-week course · No prerequisites. Full course policies, schedule, and grading breakdown.

Course description

ISM2411 introduces Python programming through a business analytics lens. Students learn the core mechanics of the language — variables, conditionals, loops, functions, data structures — and then apply them to real datasets using pandas. The course assumes no prior programming experience. By the end, students can read messy business data, clean it, summarize it, visualize it, and explain their findings.

Learning outcomes

By the end of this course, students will be able to:

  1. Write Python scripts that use variables, conditionals, loops, and functions to solve business problems
  2. Read, write, and manipulate structured data using lists, dictionaries, and the pandas library
  3. Load datasets from CSV files, clean them, and compute descriptive statistics
  4. Group, filter, and aggregate data to answer business questions
  5. Create basic visualizations (bar, line, scatter) and interpret what they show
  6. Use Git and GitHub to manage and submit code
  7. Use AI coding assistants responsibly — verify, debug, and disclose

Quizzes

Weekly quizzes appear at the end of each reading page. Quizzes marked ★ MIDTERM-ELIGIBLE (W1-8) and ★ CAPSTONE-ELIGIBLE (W10-15) contain questions that may appear verbatim on the midterm or capstone-related assessments. Study these.

Late work

Labs are due Sunday at 11:59pm ET of the week assigned. Late submissions lose 10% per day for up to 3 days, then are not accepted. One free "no-questions-asked" 48-hour extension is available per semester — request it before the deadline.

Attendance

Attendance is not graded directly but participation (5%) is. Missing more than 3 classes without explanation will affect this score. If you must miss class, watch the lecture page for that week and complete the in-class activity on your own.

AI and coding assistants

This course teaches you to use AI tools as a professional would: deliberately, with verification, and with disclosure. See the AI Use Policy for the three approved modes (Explainer, Debugger, Generator) and disclosure requirements.

Academic integrity

You may discuss concepts with classmates. You may not share code for graded labs. You may use AI tools per the AI policy. Copying classmates' code or undisclosed AI generation on the midterm or capstone is academic dishonesty and will be reported to the Office of Student Conduct.

Accessibility

Students requiring accommodations should register with USF Student Accessibility Services and provide documentation in the first two weeks. All course materials on this site work with screen readers and offer dark/light themes plus adjustable font sizes (top-right controls).
