Inventory List Manager

Build a small inventory tool using lists, indexing, slicing, and list methods. By the end, you will have a working script that manages a product inventory like a real operations tool.

Module 10 · Lab Unit 3 GitHub submission
This module: Reading Lecture Lab

Exercises

EXERCISE 1 · Beginner
Build the list
Create inventory = ["pen", "notebook", "stapler", "tape", "marker"]. Print it. Then print the total number of items using len().
Expected output: ['pen', 'notebook', 'stapler', 'tape', 'marker']  |  5
EXERCISE 2 · Beginner
Index practice
Print the first item, the last item, and the middle item using positive and negative indices. Do not hard-code the position of the middle item — compute it from len(inventory).
Expected output: pen  |  marker  |  stapler
EXERCISE 3 · Beginner
Slicing
Print the first three items using a slice. Then print the last two items using a negative slice. Then print every other item using a step slice [::2].
Expected output: ['pen', 'notebook', 'stapler']  |  ['tape', 'marker']  |  ['pen', 'stapler', 'marker']
EXERCISE 4 · Intermediate
Modify
Append "envelope". Remove "tape". Sort alphabetically. Print after each operation so you can see the list change step by step.
Final expected output: ['envelope', 'marker', 'notebook', 'pen', 'stapler']
EXERCISE 5 · Intermediate
Tuple comparison
Create coords = (40.7128, -74.0060) for NYC. Try to change coords[0] = 99 — observe the TypeError. Catch the error in a try/except block, print a message explaining why tuples are useful for coordinates, and then print the original coords unchanged.
Expected: TypeError caught, explanation printed, (40.7128, -74.006) printed
EXERCISE 6 · Intermediate
Average of a list of sales
Given sales = [100, 250, 75, 480, 200], compute and print: the total using sum(), the count using len(), and the average. Also print the best and worst day using max() and min().
Expected: Total: 1105  |  Average: 221.0  |  Best: 480  |  Worst: 75
EXERCISE 7 · Intermediate
Accumulator pattern
Given prices = [9.99, 14.99, 4.99, 24.99, 1.99], write a loop that builds a new list called discounted where every price is reduced by 10%. Use .append() inside the loop. Print both the original and discounted lists.
Expected: [8.99, 13.49, 4.49, 22.49, 1.79] (rounded to 2 decimal places)
EXERCISE 8 · Advanced
Filter above threshold
Given sales = [340, 127, 589, 204, 467, 88, 731, 315, 62, 490], build a new list high_sales containing only days where sales exceeded 300. Print high_sales, its length, and the average of the high-sales days.
Expected: [340, 589, 467, 731, 315, 490]  |  Count: 6  |  Avg: ~488.67

Stretch Challenge

Write a function inventory_report(items) that accepts a list of product names and prints a formatted report: the count of items, the alphabetically first and last products, and a sorted list. Call it with your inventory list. Then call it with a list of your own choosing to verify it works on any list.

Reflection

Before you submit

In a Python comment at the top of your file, answer these two questions in 2-3 sentences each:

  1. What is the most surprising thing about how lists work in Python compared to what you expected coming in?
  2. Describe one real business situation — from a job you've had, an internship, or a class — where a Python list would have saved you time compared to doing it manually in Excel.
