Unit 1 · Modules 1–5
Foundations
Build comfort with computers, the command line, and Python's core building blocks.
Overview Cheat Sheet
MODULE 01
What is a Computer?
CPU, memory, storage in plain English; files, paths, directories; what a program is. No coding this module — conceptual foundation only.
Reading Lecture Lab
MODULE 02
The Command Line
Terminal basics — cd, ls, mkdir, pwd. Relative vs. absolute paths. Installing Python and VS Code. Run your first script from the terminal.
Reading Lecture Lab
MODULE 03
Variables & First Output
Strings, ints, floats, booleans. print() and input(). f-strings. Brief Jupyter notebook orientation. Storing product name, price, quantity.
Reading Lecture Lab
MODULE 04
Operators & Expressions
Arithmetic, comparison, logical. Order of operations. Business formulas: revenue, margin. Integer division and type mismatches.
Reading Lecture Lab
MODULE 05
Conditionals
if / elif / else. Decision logic. Nested conditionals and when to avoid them. Discount tiers, approval thresholds, flagging anomalies.
Reading Lecture Lab
Unit 2 · Modules 6–8
Control Flow & Structure
Write programs that repeat and organize tasks; manage code with Git.
Overview Cheat Sheet
MODULE 06
Loops
for and while loops. Iterating over ranges and lists. break and continue. Process a list of sales transactions.
Reading Lecture Lab
MODULE 07
Functions, Debugging & AI Literacy
Defining and calling functions. Tracebacks, print debugging, rubber duck debugging. Debug First, Then Ask — the three modes of AI use and the disclosure rule.
Reading Lecture Lab
MODULE 08
Git & GitHub
Why version control matters. init / add / commit / push. README, folder conventions, commit messages. All future code submitted via GitHub.
Reading Lecture Lab

Module 9 — Midterm Exam
Covers Units 1–2 (Modules 1–8) · open notes · 60–75 min · no new content this module
Exam Guide
Unit 3 · Modules 10–12
Data Structures
Work with collections of data — the foundation of all business analytics.
Overview Cheat Sheet
MODULE 10
Lists & Tuples
Indexing, slicing, common list methods. When to use a tuple instead. A product inventory list.
Reading Lecture Lab
MODULE 11
Dictionaries
Key-value pairs. Nesting, adding, updating, deleting. Iterating. Customer records and lookup tables.
Reading Lecture Lab
MODULE 12
Working with Files
Reading and writing .txt and .csv files. The csv module. Relative paths in a repo. Import sales data, export a cleaned report.
Reading Lecture Lab
Unit 4 · Modules 13–15
Data Analysis with pandas
Apply Python to real business data using industry-standard tools.
Overview Cheat Sheet
MODULE 13
Intro to pandas
Excel vs. Python for data work. DataFrames and Series. Load a CSV. View, filter, sort. Explore a retail sales dataset.
Reading Lecture Lab
MODULE 14
Data Cleaning & Descriptive Stats
Missing values. Renaming columns, fixing dtypes. describe(), mean(), median(). Clean a messy sales dataset.
Reading Lecture Lab
MODULE 15
Grouping & Visualization
groupby() and aggregations. Bar, line, histogram with matplotlib. Seaborn for quick stats charts. Choosing the right chart.
Reading Lecture Lab

Capstone
Apply everything end-to-end; present findings to the class.
